This GitHub repository hosts the static web pages (HTML/CSS) that contain the legal and compliance documents required for the Intellifeed application on Facebook.

These pages are necessary to meet the Facebook platform's requirements regarding user data management and the terms of service for a third-party application.

This repository contains the following official documents for the Intellifeed application:

Privacy Policy (privacy_policy.html)

Terms of Service (terms_of_service.html)

User Data Deletion Instructions (data_deletion_instructions.html)

Intellifeed is a web application designed to help users create and publish content to a Facebook Page.
